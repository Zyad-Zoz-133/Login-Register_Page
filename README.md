# 🔐 Login & Register UI

A simple and responsive **Login & Register** user interface built using **HTML5** and **CSS3**. The project focuses on creating a clean, modern authentication design with a responsive layout.

---

## 🚀 Features

- 🔑 Login Page
- 📝 Register Page
- 📱 Responsive Design
- 🎨 Modern Dark Theme
- ✨ Smooth Hover Effects
- 💻 Clean & Organized Code
- 🎯 Built with HTML5 & CSS3 only
